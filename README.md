# ActKind v13

This is a laravel project that is currently hosted on laravel cloud.  It can be run locally and requires
* PHP 8.4+

### Font awesome kit
* to update the kit use `npm update '@awesome.me/kit-634f55b8fd'`
* then sync the kit with blade-icons  `php artisan blade-fontawesome:sync-icons --kit=634f55b8fd`
